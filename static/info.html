JSBach


JSBach es una página web diseñada para facilitar la configuración de un
sistema Linux utilizado como router y switch, permitiendo administrar
distintas funciones de red desde una interfaz sencilla.

La aplicación se organiza en varias pestañas, como WAN, NAT,
BRIDGE y FIREWALL, cada una orientada a una tarea específica
dentro de la configuración del sistema.
